# MyCrowdApp (Vue 3 + Vite)

A clean **crowdfunding template** inspired by the idea of CrowdUp — but built from scratch with **Vue 3**, **Vue Router**, and **Pinia**. It ships with a modern UI, mock data, and fully working flows: browse campaigns, view details, create, and back.

1.Download nodejs: "https://nodejs.org/en/download/prebuilt-installer"
2.Clone the repo:
3. run in powershell(terminal) :node -v
                                npm -v
4.Again run:npm install
            npm run dev
5.After step 4, it should give you a local URL (something like http://localhost:5173/ or http://localhost:3000/).
Open that in your browser to see your app running
